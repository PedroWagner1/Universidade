O que é HTML Semântica e Porque é Importante?

Html Semântica é a prática de utilizar tags HTML que possuam significado sobre seu conteúdo. Html Semântica é fundamentalmente importante pois, garante uma melhor análise de código e ranqueamento de Crawlers e tecnologias a respeito de seu conteúdo. Visualmente podemos dizer que não se aplica diferença, o diferencial verdadeiro se dá em nível de tecnologias que utilizam de análise de códigos, como crawlers e ranquemento de sites.

As tags Semânticas HTML e suas Funções

Existem diversas tags HTML Semânticas, as principais abordadas na aula foram:

  • < main > - Conteúdo Principal
  • < header > - Cabeçalho da Página
  • < footer > - Rodapé da Página
  • < article > - Conteúdo Indepente
  • < nav > - Menu de Navegação
  • < section > - Bloco de Conteúdo Temático
  • < aside > - Conteúdo Complementar (Ex: Barra Lateral)
