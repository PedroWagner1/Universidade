Vídeo: A nova era da Mídia Web

Importar vídeos em páginas web necessitava de plugins como o Flash Player antes do HTML5, causando problemas de segurança (como vulnerabilidades) e compatibilidade.

  • CVE-2015-5119
  • CVE-2015-5119
  • CVE-2013-5331
  • CVE-2010-1297
    • Cve's de Vulnerabilidades conhecidas no Flash Player da época.

O elemento video mudou o cenário da aplicação de multimídia em páginas web, oferecendo reprodução nativa de vídeo em todos os navegadores da época.

Implementação Básica de Vídeo:

Os elementos configurados nestga implementação foram:

  • video
    • width = 600
    • height = 400
    • controls (define controles de reproduição)
    • poster (define imagem pré visualização)
  • source (para indicar o arquivo mp4)

Atributos Essenciais do Elemento Video:

  • controls - Controles de Reprodução
  • autoplay - Executa vídeo automaticamente (vem com muted em muitos clientes)
  • loop - Vídeo fica em loop contínuo
  • muted - Omite o áudio do vídeo
  • preload - define estratégia de carregamento
  • poster - define imagem de pré visualização

Métodos Javascript para Controle de Vídeos:

Alguns métodos existentes em javascript para controle de vídeo são:

const video = document.getElementById('video');
// Métodos Principais:

video.play();
video.pause();
video.load();
// Propriedades Úteis:

video.currentTime = 30; // Pula 30 segundos
video.volume = 0.5; // Volume em 50%
video.playbackRate = 1.5; // Vídeo em velocidade 1.5x

Eventos de Vídeo:

video.addEventListener('loadstart', () => console.log('Iniciou Carregamento'))
video.addEventListener('canplay', () => console.log('Pronto para Reprodução'))
video.addEventListener('ended', () => console.log('Reprodução finalizada.'))
video.addEventListener('error', () => console.log('Erro na reprodução'))
